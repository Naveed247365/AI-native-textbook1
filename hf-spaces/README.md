# AI-native Textbook Platform for Physical AI & Humanoid Robotics

This is a FastAPI backend with RAG functionality running on Hugging Face Spaces.

## Endpoints
- `/health` - Health check
- `/api/chat` - Chat functionality
- `/api/*` - Other API endpoints

## Features
- AI-powered textbook assistance
- RAG (Retrieval Augmented Generation)
- OpenRouter integration
- Qdrant vector database
