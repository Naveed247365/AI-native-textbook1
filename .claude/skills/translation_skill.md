# Translation Skill

## Purpose
This skill handles real-time translation of textbook content between English and Urdu, maintaining technical accuracy and educational value.

## Capabilities
- Translate content between English and Urdu
- Preserve technical terminology accuracy
- Maintain code comments and documentation during translation
- Handle cultural adaptation for Urdu-speaking users
- Provide quality assurance for translations

## Usage Context
- Called when users request Urdu translation of content
- Used by the "اردو میں دیکھیں" translation button
- Integrated with the translation service
- Used for bidirectional translation (English ↔ Urdu)

## Input Format
- Content to be translated
- Source language (en/ur)
- Target language (en/ur)
- Context information (chapter, section)

## Output Format
- Accurately translated content
- Preserved technical terminology
- Culturally appropriate language
- Quality score or confidence level

## Constraints
- Must maintain technical accuracy of robotics/AI concepts
- Should preserve educational value of original content
- Must follow proper Urdu script and grammar
- Should maintain consistency in technical terminology
- Must achieve quality threshold (BLEU score ≥ 0.7 or equivalent)
