# Textbook Navigation Skill

## Purpose
This skill helps users navigate the Physical AI & Humanoid Robotics textbook effectively, guiding them through the content structure and helping them find relevant information.

## Capabilities
- Identify relevant chapters and sections based on user queries
- Guide users through the logical progression of topics
- Connect related concepts across different modules
- Suggest prerequisite knowledge for advanced topics
- Help users find specific technical information

## Usage Context
- Called when users need help finding information in the textbook
- Used to suggest next steps in learning progression
- Integrated with the chatbot for navigation assistance
- Used to connect concepts across modules

## Input Format
- User's query about textbook content
- Current location in the textbook
- User's experience level
- Specific topic or concept they're looking for

## Output Format
- Specific chapter/section recommendations
- Learning pathway suggestions
- Cross-references to related content
- Prerequisite information when needed

## Constraints
- Must reference actual textbook content and structure
- Should guide users to appropriate content for their level
- Must maintain the "Embodied Intelligence" theme connection
- Should support both linear and non-linear navigation
