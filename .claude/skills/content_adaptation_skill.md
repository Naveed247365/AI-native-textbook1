# Content Adaptation Skill

## Purpose
This skill adapts textbook content in real-time based on user preferences, background, and experience level to provide a personalized learning experience.

## Capabilities
- Modify content difficulty dynamically
- Adjust examples to match user background (software vs hardware focus)
- Adapt explanation depth based on experience level
- Provide alternative explanations when needed
- Adjust technical terminology based on user familiarity

## Usage Context
- Called when users request personalized content
- Used when displaying chapters to registered users
- Integrated with the personalization service
- Called when users change their preference settings

## Input Format
- Original content to be adapted
- User profile information (background, experience level)
- Current chapter/module context
- User's preferred focus areas

## Output Format
- Adapted content matching user profile
- Modified examples and explanations
- Adjusted difficulty level
- Personalized terminology and concepts

## Constraints
- Must maintain educational value of original content
- Should not oversimplify critical concepts
- Must respect user's stated preferences
- Should enhance comprehension without losing technical accuracy
- Must align with the "Embodied Intelligence" framework
